# Flipkart Login Module

A frontend web application for a Flipkart-like login module built with Java.

## Project Overview

This project implements a login and authentication module for an e-commerce platform similar to Flipkart.

## Features

- User login functionality
- User registration
- Password management
- Session management
- Form validation
- Responsive design

## Technology Stack

- **Backend**: Java
- **Frontend**: HTML5, CSS3, JavaScript
- **Build Tool**: Maven/Gradle (to be added)

## Getting Started

### Prerequisites

- Java 8 or higher
- Maven 3.6 or higher

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/prashantrathod-tech/flipkart-login-module.git
   cd flipkart-login-module
   ```

2. Build the project:
   ```bash
   mvn clean build
   ```

3. Run the application:
   ```bash
   mvn spring-boot:run
   ```

## Usage

- Navigate to `http://localhost:8080` in your browser
- Enter your credentials or create a new account
